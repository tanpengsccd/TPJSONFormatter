# TPJSONFormatter
A Mac App for  [HandyJSON](https://github.com/alibaba/HandyJSON)    
# How to use?
1. APP放入应用程序目录,先打开一次，系统偏好设置-扩展-勾选TPJSONFormatter，Xocde-Preference-Key Bindings-搜索“TPJSONFormatter” 设置好快捷键。            
使用复制JOSN 到粘贴板，使用快捷键，即可生成代码


2. 直接打开APP
